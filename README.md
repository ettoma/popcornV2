# popcorn

A watchlist tracker in Flutter with Golang backend

## What does it do

- Look up movies and add them to your watchlist.
- Rate them after watching to create your personal movie archive


## Tech stack

Flutter + Dart
Golang REST API
Postgres database
Firebase authentication
